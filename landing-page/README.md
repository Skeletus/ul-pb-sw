# WorkMeter Landing Page

Sitio publico autonomo para WorkMeter, construido con Next.js App Router, TypeScript y Tailwind CSS.

## Requisitos

- Node.js 18.18 o superior.
- npm.

## Correr localmente

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.

## Build

```bash
npm run build
```

El proyecto exporta contenido estatico y puede publicarse en Vercel, Netlify u otro hosting compatible.

## Contenido

La pagina comunica WorkMeter como una plataforma para constructoras que necesitan monitorear maquinaria alquilada, detectar inactividad, generar alertas y calcular costos desperdiciados mediante sensores y reportes.
